# Windows E2E 手動測試清單

本清單供在 **Windows 10/11** 上，搭配已安裝的 **Palworld 1.0 專用伺服器** 進行端到端驗證。建議使用 GUI 安裝包或免安裝版，並在測試前備份 `Pal/Saved/SaveGames`。

## 測試前準備

- [ ] 確認 Windows 使用者名稱與安裝路徑均為 ASCII（無中文/特殊字元）
- [ ] 防火牆已放行：`8211/UDP`（遊戲）、`8212/TCP`（REST API）、`27015/UDP`（社群列表，若需要）
- [ ] 以系統管理員身分執行 GUI（首次 SteamCMD 安裝時建議）
- [ ] 記錄測試版本：GUI `1.2.1`、引擎版本、Palworld 客戶端版本
